#  Professor Assistant — Python Exam Generator

A Python program that helps professors generate random exams from a question bank file.

## Features
✔ Reads question-answer pairs from text file  
✔ Generates random exam based on required number of questions  
✔ Saves generated exam into separate file  
✔ Allows multiple exam creation in one runtime session  

## Run the program

```bash
python professor_assistant.py
